# Аналіз E-commerce даних Google Analytics 4 в Google BigQuery

## Опис проєкту
Аналіз сирих event-based даних Google Analytics 4 (на прикладі публічного датасету інтернет-магазину Google Ecommerce) у хмарному сховищі Google BigQuery для дослідження поведінки користувачів, сегментації клієнтської бази та оцінки ефективності продажів.

---

## Що було зроблено в межах проєкту:

* **Роботи з Google Analytics 4 даними:** Проведено аналіз структуру подій, параметрів та користувацьких властивостей для отримання інсайтів про поведінку відвідувачів сайту.
* **Аналіз e-commerce метрик:** Досліджено товарний асортимент, розраховано дохід від продажів, виявлено найприбутковіші товарів та категорії для оптимізації товарної стратегії.
* **Сегментація користувачів:** Реалізовано ранжування клієнтів за витратами, виявлено VIP-користувачів та проаналізована їх поведінк для створення персоналізованих маркетингових кампаній та програм лояльності.
* **Робота з великими датасетами (BigData):** Оптимізовано запити через використання партицій, проведена робота з вкладеними структурами даних та масивами для швидкого отримання результатів на великих обсягах даних

---

## Технології:
* **СУБД:** Google BigQuery (SQL)
* **Датасет:** `bigquery-public-data.ga4_obfuscated_sample_ecommerce.events_*`
