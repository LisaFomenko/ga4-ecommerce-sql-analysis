# Аналіз E-commerce даних Google Analytics 4 в Google BigQuery

## Опис проєкту
Аналіз сирих event-based даних Google Analytics 4 (на прикладі публічного датасету інтернет-магазину Google Ecommerce) у хмарному сховищі Google BigQuery для дослідження поведінки користувачів, сегментації клієнтської бази та оцінки ефективності продажів.

---

## Що було зроблено в межах проєкту:

* **Робота з event-based моделлю GA4:** Досліджено структуру подій, параметрів та користувацьких властивостей. Це дозволило отримати глибинні інсайти про те, як саме відвідувачі взаємодіють із сайтом.
* **Аналіз e-commerce метрик:** Розраховано чистий дохід від продажів, а також виявлено найприбутковіші товари та категорії для оптимізації товарної стратегії.
* **Сегментація користувачів:** Реалізовано аналіз VIP-клієнтів через ранжування користувачів за їхніми сумарними витратами. Це допомагає бізнесу знаходити найціннішу аудиторію для запуску персоналізованих маркетингових кампаній.
* **Аналіз користувацьких сесій:** Оптимізовано запити для визначення тригерів, які найчастіше стають стартовими для початку сесії користувача на сайті.
* **Робота з великими датасетами (BigData):** Оптимізовано SQL-запити через використання партицій (`_TABLE_SUFFIX`), реалізовано ефективну роботу з вкладеними структурами даних та масивами (`UNNEST`, `STRUCT`, `ARRAY_LENGTH`) для швидкого отримання результатів на великих обсягах даних.

---

## Технології:
* **СУБД:** Google BigQuery (SQL)
* **Датасет:** `bigquery-public-data.ga4_obfuscated_sample_ecommerce.events_*`
